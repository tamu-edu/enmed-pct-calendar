EnMed Calendar

A lightweight academic calendar application for the Texas A&M Engineering Medicine (EnMed) program.

Built as a fully static GitHub Pages app using:

HTML / CSS / JavaScript; 
events.json for event storage; 
GitHub API for live editing; 
SheetJS for bulk Excel uploads

Features:
Week, month, and list views;
Cohort and category filtering;
Faculty editing mode;
Bulk XLSX upload;
GitHub-backed shared updates
